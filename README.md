# AI-Based Network Intrusion Detection System

## Description
This project demonstrates an AI-Based Network Intrusion Detection System that detects malicious network traffic using machine learning. The system is deployed as a web-based dashboard and simulates live network packet analysis.

## Features
- AI-based intrusion detection
- Random Forest machine learning model
- Live packet simulation
- AI-generated explanations
- Web dashboard deployment
